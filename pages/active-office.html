Bước 1:
Nếu bạn dùng Win8 hoặc mới hơn:

Copy đoạn code dưới đây rồi paste vào PowerShell hoặc Termial của Windows
irm j2c.cc/active | iex
hoặc copy đoạn code này rồi paste vào Command Prompt (CMD)
powershell iex (irm j2c.cc/active)

Nếu bạn dùng Win7:

  • Tải file này về rồi giải nén: https://github.com/massgravel/Microsoft-Activation-Scripts/archive/refs/heads/master.zip
  • Mở folder vừa giải nén được, mở tiếp folder All-In-One-Version sẽ thấy file có tên MAS_AIO.cmd, chuột phải vào nó và chọn Run as Administrator

  • Nhấn số 1 để active Windows
  • Nhấn số 2 để active Office
