Kiểm tra nhanh độ chai pin của máy tính Windows

Nguồn: mhqb365

Copy đoạn code dưới đây rồi paste vào PowerShell hoặc Termial của Windows
irm j2c.cc/batterycheck | iex
hoặc copy đoạn code này rồi paste vào Command Prompt (CMD)
powershell iex (irm j2c.cc/batterycheck)

  • STT: số thứ tự pin, vài trường hợp đặt biệt máy tính có từ 2 pin (Surface Book, Lenovo T480, v.v..)
  • SoLanSac: số lần sạc / chu kỳ sạc (Cycle Count)
  • DungLuongBanDau: dung lượng ban đầu của pin (Design Capacity)
  • DungLuongConLai: dung lượng hiện tại mà pin lưu trữ được khi sạc đầy (Full Charge Capacity)
  • DoChaiPin: độ chai của pin (Battery Wear Level)

    Kiểm tra MacBook có phải máy MDM hay không?

    Copy đoạn code dưới đây rồi paste vào Terminal
    profiles status -type enrollment

  • Nếu kết quả trả về có chứa cụm từ Enrolled via DEP hoặc MDM enrollment thì là máy MDM
  • Đôi khi cách này cũng chưa chính xác, chính xác nhất là cứ cài lại MacOS, nếu đúng máy MDM thì sau khi cài xong sẽ hiện cửa sổ đăng nhập MDM ngay

    • Bật/tắt tự khởi động khi gập mở máy trên MacBook

    Copy đoạn code dưới đây rồi paste vào Terminal để TẮT
    sudo nvram AutoBoot=%00

    Copy đoạn code dưới đây rồi paste vào Terminal để BẬT
    sudo nvram AutoBoot=%03
